Tips for fetching Sacramento's farm2fork data from OSM:

Each location point in OSM is called a node. Nodes can have one or more key-value pairs which describe what they are.

interest OSM key-value pair
eating and drinking places amenity=restaurant
amenity=cafe
amenity=fast_food
amenity=pub
amenity=bar
coffee shops use shop=coffee for a shop that specialises in selling the ingredients, amenity=cafe for one which sells the drink
farmers markets shop = farm
places that offer locally-sourced food/ingredients local_food=yes
places that offer organic food/ingredients organic_food=yes
places that offer vegan food/ingredients vegan_food=yes
places that offer gluten-free food/ingredients gluten-free_food=yes

For more technical information about fetching and using data from OpenStreetMap, check out this very helpful site.
